{# --- Stripe-style dense settings components (FoodAssistant-jcnh) ---------
   A reusable panel/row/pill/toggle vocabulary so a settings section reads as a
   tight table of labelled controls rather than a stack of big cards. The long
   paragraphs of helper text become info() tooltips, leaving a one-line label
   plus a short hint. All scoped under .setup-wrap in setup.html so the classes
   never leak onto the /ui pages. #}

{# A small state chip. kind: good (green), neutral (muted), pink (accent),
   warn (amber). pill_id lets a caller update it live (e.g. auth on/off). #}
{% macro pill(text, kind='neutral', pill_id='') -%}
{{ text }}
{%- endmacro %}

{# A toggle switch (pink when on). Renders a plain checkbox with the given id so

   disabled locks it read-only on a satellite (server-managed settings). #}
{% macro toggle(id, checked=False, onchange='', label='', disabled=False) -%}

{%- endmacro %}

{# A bordered panel with a header (title + optional one-line desc + optional

    {{ caller() }}
  



{%- endmacro %}
