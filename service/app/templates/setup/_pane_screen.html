Kiosk Display

Scale and CSS rotation apply immediately in the kiosk browser without a reboot.

Larger for a small panel; smaller for a wide monitor.
Pick the type that matches your panel. A resistive Waveshare 3.5-4 inch HDMI LCD is the ADS7846 SPI option; a Waveshare HDMI HAT with a USB touch cable is the USB touch option. After choosing, use "Apply touch driver" below so the boot overlay is written (a reboot loads it).
{% if is_pi %}
Writes the boot config the selected display type needs (for an ADS7846 SPI panel this enables SPI and adds the touch overlay). Needed once, and again only if you change the display type. Takes effect after a reboot.
Launches a four-corner tap test on the Pi's attached touchscreen. Stand at the device and tap each crosshair. If a calibration comes out wrong, Reset reverts to the panel default. Recalibrate after changing the display rotation.
{% endif %}
Blanks the kiosk screen after inactivity. A touch, key press, or a Stream Deck button press wakes it. The Stream Deck has its own separate timeout. 0 disables.
Wakes a sleeping screen when the device is moved or bumped, using the built-in accelerometer on kits that include one. A screen touch or a Stream Deck button press always wakes the display, with or without this. Auto keeps it off on devices without the sensor.
Dims the page to the gliding logo and clock after inactivity; any touch brings the page right back. Unlike Display sleep above, which switches the panel off, the screensaver keeps the display on, so it suits panels that wake slowly or misbehave when powered off. This is a per-device choice.

Photos plays images from a USB drive's photos folder and falls back to the logo when no drive is present. Put your pictures in a folder named photos or pictures at the top of the drive. This is a per-device choice.
Test starts the screensaver on this screen right away with the style and speed picked above; any touch or key press dismisses it. To preview a kiosk panel, open its own settings page and press Test there.

On-screen navigation bar

An optional fixed bar of nav icons docked to a screen edge, handy on touch screens. It reserves space so it never overlaps the page. Bottom is a horizontal bar; left and right are vertical. The setting below is the default for devices that haven't been customised on this device.
