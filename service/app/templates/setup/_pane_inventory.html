Grocy: Inventory Backend

Grocy is the open-source inventory app that stores your stock, storage locations, and consumption history. Pantry Raider reads and updates it through Grocy's API, so it needs the Grocy address and an API key. {% if is_pi %}Grocy is running on this device at port 9383.{% else %}Connect to a Grocy instance running on your network or server.{% endif %}
