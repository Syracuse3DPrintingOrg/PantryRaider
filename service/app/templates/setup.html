Barcode Enrichment Optional

Open Food Facts data is messy ("zero sugar" for Dr Pepper Zero). With enrichment on, a small text-only LLM call per scan cleans up the name, category, storage location, and shelf-life estimate. Falls back to heuristics if the call fails.

Uses the API key / base URL from the Vision section above. Pick Ollama here to keep scans fully local while using a cloud provider for photos.
Enrichment is text-only, so a small/cheap model works well (e.g. claude-haiku-4-5, gpt-4o-mini, llama3.2:3b).
