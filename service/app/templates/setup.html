    Barcode Enrichment

    Open Food Facts names are often messy ("zero sugar" for Dr Pepper Zero). LLM enrichment cleans up the name, category, storage, and shelf life per scan — one small text call.

    When a barcode isn't in Open Food Facts, ask the AI to identify it by barcode number. Results are low-confidence guesses — the item goes to the pending queue for review. Only useful with LLM enrichment enabled.
    Uses the key/URL from the Vision section. Pick Ollama to keep scans fully local.
    Text-only, so a small model works well (claude-haiku-4-5, gpt-4o-mini, llama3.2:3b).

    Cloudflare Tunnel (free, DIY) — runs a cloudflared container on your host. FoodAssistant Cloud — managed subscription with a stable URL. Both modes require Docker to be available on the host.

    Get your token from the Cloudflare Zero Trust dashboard — create a tunnel, copy the token. Enter the subscription token from your FoodAssistant Cloud account.

    Save your data off-box and keep the app up to date.

    Backup

    FoodAssistant's data directory (settings, database, staples list) as a zip. Grocy and Mealie data live in separate containers — use scripts/backup.sh on the host for a full snapshot.

    By default secrets are stripped (and rclone.conf is omitted) so the file is safe to store off-box; after restoring you re-enter keys here. Only include them if the destination is trusted.
    Download Backup
    Any rclone remote you've configured via docker exec foodassistant-service rclone config. Config is stored at /app/data/rclone.conf.
