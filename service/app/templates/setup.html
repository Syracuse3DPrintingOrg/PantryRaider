     Backup
    
            
    FoodAssistant's data directory (settings, database, staples list) as a zip.
              Grocy and Mealie data live in separate containers — use
              scripts/backup.sh on the host for a full snapshot.
    
            
    
              
              
              
    By default secrets are stripped (and rclone.conf is omitted) so the
                file is safe to store off-box; after restoring you re-enter keys here. Only include them if the
                destination is trusted.
    
            
    
            
    
              
                Download Backup
              
              
            
    
            
    
              
              
    
                
                
              
    
              
    Rewrites this app's data (settings, database, staples) from the zip. The
                current data is copied aside first, and a redacted backup keeps the API keys you already have.
                Grocy and Mealie data are not touched here. Reload the page after a restore.

     Barcode Scanner
    
            
    A USB or Bluetooth scanner plugged into any device (this server, a Pi host, or a Pi Remote) works on the Add Food page automatically, all at once, alongside the in-browser camera scanner. This setting is just a hint for which method to reach for first; it does not turn the others off.
